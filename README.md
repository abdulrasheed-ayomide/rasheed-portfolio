## Installation

Clone the repository:

```bash
git clone https://github.com/abdulrasheed-ayomide/rasheed-portfolio.git
```

Navigate to the project:

```bash
cd rasheed-portfolio
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
